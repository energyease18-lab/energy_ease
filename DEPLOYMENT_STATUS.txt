================================================================================
                           TROUBLESHOOTING GUIDE
================================================================================

IF IMAGES DON'T SHOW ON LIVE SITE:

1. Hard Refresh Browser:
   - Windows/Linux: Ctrl + Shift + R
   - Mac: Cmd + Shift + R

2. Clear Browser Cache:
   - Open DevTools (F12)
   - Right-click refresh button
   - Choose "Empty cache and hard refresh"

3. Check Browser Console (F12):
   - Look for red error messages
   - Note any 404 errors for specific image files
   - Check Network tab to see if images are being requested

4. Verify Deployment:
   - Check that dist/ folder was published
   - Confirm dist/assets/ contains all 8 JPEG images
   - Verify _redirects file exists in dist/

5. Contact Bolt Support if:
   - Images still don't load after hard refresh
   - Console shows 404 errors for images
   - Other pages fail to load
